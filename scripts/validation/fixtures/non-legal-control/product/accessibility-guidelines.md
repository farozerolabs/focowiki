# Product Accessibility Guidelines

Interactive controls require visible focus states, descriptive labels, and keyboard access. Color cannot be the only signal for status or validation.

Content authors should use informative headings and link text. Error messages must identify the affected field and the action required to continue.

Release reviews apply these checks before following the [Change Management Procedure](../operations/change-management.md).
