# Atlas Version 2 Release Notes

Version 2 adds project-level export permissions and a searchable audit history. Workspace owners can grant export access without granting editing access.

The audit history records configuration changes for 180 days. Existing workspaces receive the new history automatically after the upgrade.

Deployment teams should follow the [Change Management](../../operations/change-management.md) checklist before enabling export permissions.
