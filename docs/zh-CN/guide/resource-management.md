# 知识库资源管理

Admin UI 支持修改知识库信息和维护已上传的 Markdown 资源。

## 知识库信息

打开知识库卡片菜单并选择**修改**，可以更新知识库名称和描述。卡片和详情页侧边栏会立即使用保存后的信息。发布完成后，生成的导航文件会使用更新后的信息。

## 文件

打开文件树中的文件菜单，可以重命名文件、将文件移动到另一个已上传目录、替换完整 Markdown 正文或删除文件。文件名必须以 `.md` 结尾。正文替换支持直接编辑文本，也支持选择本机 `.md` 文件。

文件完成重命名、移动和正文替换后会保留稳定的来源文件 ID。修改处理期间，当前生成内容仍可正常读取。

## 目录

打开目录菜单，可以重命名、移动或删除目录。目标目录选择器会在打开目录时加载已上传的来源目录。当前目录及其下级目录不能作为移动目标。

目录修改会在发布完成后更新全部下级路径。删除目录会包含全部下级来源文件。

## 并发修改

资源存在已受理的修改时，Admin UI 会停用对应菜单。资源版本过期、目标路径已占用、文件正在处理或资源正在删除时，页面会显示明确提示。资源版本过期时需要刷新文件树后重试。目标路径已占用时需要选择其他目录或名称。

Admin UI 与 Developer OpenAPI 使用相同的资源 ID、版本校验、操作状态和最终生成文件树。
